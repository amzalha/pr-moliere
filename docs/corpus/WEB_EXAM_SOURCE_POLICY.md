# PR_Molière — Politique d’intégration des examens web

## Statut actuel

Les examens réels provenant d’AlloSchool, Moutamadris ou Attarbia ne sont pas copiés dans PR_Molière.

PR_Molière contient actuellement :

- des sujets modèles originaux ;
- un catalogue de liens vers les plateformes éducatives ;
- une structure prête à accueillir des sujets autorisés ou reformulés.

## Règle d’intégration

Ne pas copier automatiquement :

- PDF ;
- sujets complets ;
- corrections ;
- barèmes ;
- textes protégés.

Utilisation autorisée à ce stade :

- liens externes ;
- métadonnées ;
- niveau ;
- type d’évaluation ;
- source ;
- région ;
- année ;
- thème général.

## Méthode recommandée

Pour enrichir légalement le corpus :

1. référencer la source ;
2. créer une fiche de métadonnées ;
3. reformuler un sujet original inspiré du programme ;
4. éviter la copie exacte ;
5. conserver la traçabilité ;
6. intégrer seulement les contenus explicitement autorisés.

## Conclusion

La Phase 5 doit intégrer les sources web comme catalogue de références, pas comme extraction brute.
