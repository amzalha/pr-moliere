# PR_Molière — Phase 6
## Extension du corpus original d’examens

## Objectif

Faire passer le corpus actif de 4 sujets modèles à 30 sujets originaux structurés.

## Répartition cible

### 1AC

- 5 contrôles continus
- compétences : lecture, langue, production écrite

### 2AC

- 5 contrôles continus
- compétences : lecture, langue, portrait, dialogue, récit

### 3AC

- 10 contrôles continus
- 5 examens locaux
- 5 examens régionaux modèles

## Total cible

30 sujets originaux PR_Molière.

## Règle juridique

Les sujets doivent être originaux.

Ne pas copier :

- PDF externes ;
- textes protégés ;
- corrections de sites ;
- barèmes publiés par des plateformes externes.

Les sources web restent des références externes.

## Structure obligatoire par sujet

Chaque sujet doit contenir :

- id unique ;
- niveau ;
- type d’évaluation ;
- titre ;
- thème ;
- compétence ;
- texte support original ;
- consigne ;
- questions ;
- réponses modèles ;
- critères de correction ;
- barème total sur 20 ;
- sourceType = modele_original ;
- sourceNom = PR_Molière — sujet modèle original.

## Étapes de réalisation

1. Ajouter 6 sujets originaux supplémentaires.
2. Valider le corpus.
3. Tester l’interface Examens.
4. Ajouter ensuite des lots de 5 à 10 sujets.
5. Créer un checkpoint à chaque lot stable.

## Prochaine étape technique

Créer le premier lot : 6 nouveaux sujets originaux.
