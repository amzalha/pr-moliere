# PR_Molière — Statut d’origine du corpus

Le corpus actif contient actuellement 4 sujets modèles originaux PR_Molière.

Ces sujets ne sont pas copiés depuis AlloSchool, Moutamadris ou Attarbia.

Les sites web sont seulement référencés comme sources externes.

Un sujet web ne doit entrer dans le corpus actif que s’il est autorisé, reformulé ou transformé en sujet original.
