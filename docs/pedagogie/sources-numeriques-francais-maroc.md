# PR Molière — Sources numériques marocaines pour le français collège

## Objectif

Ce document fixe le référentiel de sources numériques marocaines utilisé pour renforcer le MVP pédagogique de PR Molière en français collège : 1AC, 2AC et 3AC.

## Sources principales

### 1. Attarbia.net

Attarbia.net sert de source de travail pour les épreuves types, les examens locaux et les examens régionaux de français.

Liens de travail :

- Examens régionaux français 3AC : https://fr.attarbia.net/p/examens-regionaux-francais-3ac.html
- Examens normalisés locaux 3AC : https://fr.attarbia.net/p/francais-3eme-annee-college-examens.html

Usage pédagogique prévu :

- repérer les structures récurrentes des examens ;
- enrichir les entraînements 3AC ;
- comparer les consignes et les barèmes ;
- préparer les élèves aux parties compréhension, langue et production écrite.

### 2. 9rayti.com

9rayti.com sert de source complémentaire pour suivre les archives et les dossiers d'examen régional marocain.

Lien de travail :

- https://www.9rayti.com/dossier/examen-regional-maroc

Usage pédagogique prévu :

- compléter les archives ;
- comparer les thèmes d'examen ;
- renforcer la préparation au brevet collégial.

### 3. Guides PDF complémentaires

Certains guides PDF peuvent servir à repérer des variantes d'épreuves, des transformations syntaxiques et des critères de production écrite.

Lien de travail :

- https://fr.scribd.com/document/739292632/Entrainement-a-l-examen-regional-3eme-annee-collegial-1

Usage pédagogique prévu :

- identifier des exercices types ;
- renforcer la correction de la production écrite ;
- améliorer les grilles de critères.

## Structures pédagogiques prioritaires

### Tableau d'énonciation

Structure importante pour les situations de communication.

Exemple :

- situation : demander poliment une autorisation ;
- énoncé attendu : formulation correcte, polie et adaptée au destinataire.

Usage dans PR Molière :

- générer des situations de communication ;
- vérifier la politesse, le registre et l'adéquation de l'énoncé ;
- entraîner les élèves aux actes de parole.

### Alternance imparfait / passé simple

Structure essentielle pour le conte, le récit et la nouvelle.

Règle pédagogique :

- l'imparfait décrit le cadre, les habitudes ou les actions durables ;
- le passé simple raconte les actions brèves, achevées et importantes.

Usage dans PR Molière :

- créer des exercices de conjugaison contextualisés ;
- corriger les valeurs des temps ;
- préparer les élèves aux textes narratifs.

## Statut

Ces sources constituent un référentiel de travail. Les contenus doivent être utilisés avec prudence : reformulation, respect des droits, traçabilité et vérification régulière des liens.
