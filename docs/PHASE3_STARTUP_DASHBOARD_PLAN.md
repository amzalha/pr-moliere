# PR_Molière — Phase 3 : Dashboard avancé + dossier startup

## Objectif général

Transformer PR_Molière d'un MVP technique fonctionnel en MVP startup présentable à un incubateur, une école privée, un centre de soutien scolaire ou un partenaire EdTech.

## Modules prévus

### 1. Dashboard avancé élève

- Nombre total de réponses
- Nombre de réponses justes
- Taux de réussite
- Progression par niveau : 1AC / 2AC / 3AC
- Progression par thème
- Dernières réponses
- Erreurs fréquentes
- Recommandations de révision

### 2. Analyse pédagogique

- Identifier les thèmes faibles
- Identifier les types d’erreurs répétées
- Générer des conseils personnalisés
- Proposer les prochaines fiches à réviser

### 3. Dossier startup

- Problème ciblé
- Solution proposée
- Public cible
- Marché potentiel
- Proposition de valeur
- Modèle économique
- Architecture technique
- Roadmap produit
- Risques
- Besoins de financement
- Plan de test avec élèves

### 4. Indicateurs startup

- Nombre d’utilisateurs testeurs
- Nombre de réponses corrigées
- Taux de rétention
- Taux de réussite moyen
- Matières / thèmes les plus utilisés
- Coût moyen IA par correction
- Potentiel abonnement

## Résultat attendu de la Phase 3

À la fin de cette phase, PR_Molière devra être présentable comme :

- MVP EdTech fonctionnel
- Projet startup sérieux
- Plateforme IA pédagogique traçable
- Produit prêt pour test terrain avec élèves
