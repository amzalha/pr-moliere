# Dossier Startup — PR_Molière

## 1. Résumé exécutif

PR_Molière est une plateforme EdTech intelligente destinée aux élèves du collège marocain, principalement pour l’apprentissage du français en 1AC, 2AC et 3AC.

L’application propose un tuteur virtuel capable de générer des exercices, corriger les réponses des élèves, donner un rappel de cours, suivre la progression et enregistrer l’historique pédagogique dans Supabase.

Le projet est déjà déployé en production avec Railway, Firebase Authentication, OpenRouter et Supabase.

## 2. Problème identifié

Beaucoup d’élèves du collège rencontrent des difficultés en français :

- compréhension des textes ;
- grammaire ;
- conjugaison ;
- production écrite ;
- préparation à l’examen régional ;
- manque d’accompagnement personnalisé ;
- coût élevé du soutien scolaire classique ;
- absence de suivi précis des erreurs.

Les parents et enseignants manquent souvent d’outils simples pour suivre les progrès réels de l’élève.

## 3. Solution proposée

PR_Molière propose un assistant pédagogique IA qui :

- explique les leçons ;
- génère des exercices adaptés ;
- corrige les réponses ;
- donne un feedback motivant ;
- enregistre l’historique ;
- calcule le taux de réussite ;
- identifie les thèmes faibles ;
- recommande les révisions prioritaires.

L’objectif n’est pas seulement de donner une réponse, mais de construire un suivi pédagogique durable.
